# 📊 Proyecto Catálogo de Cuentas

Este proyecto permite gestionar un catálogo de cuentas contables por empresa y año, con autenticación y balance general.

## 🚀 Tecnologías

- Next.js (Frontend)
- Express.js (Backend)
- MySQL (Base de datos)
- Tailwind CSS

---

## ⚙️ Requisitos

- Node.js v18+  
- MySQL Server  
- Git
